> My Shop App

Aplicación de compras en React Native para "Escuelita de React" de ASAP consulting

> Especificaciones del proyecto:

librerias usadas:
para seleccionar la fecha de nacimiento:
npm i @react-native-community/datetimepicker

Para seleccionar la nacionalidad:
$ npm install @react-native-picker/picker --save

navegación:
npm install --save react-navigation
npm install --save react-navigation-stack
expo install react-native-gesture-handler react-native-reanimated
react-native-screens react-native-safe-area-context @react-native-community/masked-view
